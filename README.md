# react-draw

Canvas based drawing input

## Installation

```
npm install react-draw
```

## Demo

http://cezary.github.io/react-draw/

## Example

```javascript
var React = require('react');
var Draw = require('react-draw');

var Component = React.createClass({
  render: function() {
    return (
      <Draw />
    );
  }
});
```
